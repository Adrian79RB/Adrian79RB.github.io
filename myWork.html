My Works

Wave Function Collapse

This is a game where I developped a system to procedurally generate the ground of the map and the decisión tree of each NPC using the same algorithm. This system let us create a different experience in each run. Watch the code in Github.

Puddies

This is a game where I developped different systems such as the tutorial system, the player tracking system, the music & sound managers, etc. The game is a relaxed one focus on the exploration, the puzzle solving and the herd management. Download the demo from Steam.

The Antidote

This is a point&click puzzles game where I developped a system that let us organize the narrative events and the unlocking content system based on the player progress making easy to the design team the process to stablish when the actions are thrown to the player. On the other hand I developped different puzzle games based on traditional games. Wach the code in Github.

UJImon

This is a pokemon-like fighting game where I developped a system to select a team to fight with different types of creatures with different abilities and attacks that are based on random selection. On the other hand I designed a AI agent that selects a team and make the movements during the fighting bearing in mind the different types of the creatures, the power of the available attacks and the relations among the types to decide the best move to win the fighting. Wach the code in Github.

John Lemon

This is a infiltration type game where I developped a system of pathfinding from scratch using A*. For creating the nodes structure I use waypoints interconnected among them. Moreover, I have design three types of enemies: one static used to protect essential positions in the map, one that is patrolling over a certain zone of the map, and the last one who is alerted when the player is found and go staright to the point where the player is. Wach the code in Github.

Input Keeper

This is an application use to record the inputs that the player is performing while the game is working. Then, the application gives the user the possibility to repeat the same inputs in the same order whenever it will decide. Wach the code in Github.
