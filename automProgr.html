Automation Programming

I have experience working in the development of tools that make repetitive tasks automatically.
Most of this experience is working with the Jenkins system, so I have used Jenkins to make automatic
integration of changes based on the commits performed in the source control system. However, I have develop
other tools, I have some experience working with C# in Windows Forms IDE, where I have developed some apps
that do some external jobs easier such as QA and Testing for instance, an app to repeat certain kinds of inputs
performed by the tester. On the other hand, I have used Unreal Engine to create some agents that can be
used to perform repetitive actions or movements and find some problems in the design of the game.

All these apps and projects are oriented to make certain repetitive aspects that could waste some valuable time
of the team, so this lets the rest of the team focus on more key aspects of the game development making all the work
more efficiently.
